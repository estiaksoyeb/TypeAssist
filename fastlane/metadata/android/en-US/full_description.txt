TypeAssist is an AI-powered accessibility assistant that works directly within any Android text field. It leverages the Accessibility Service API to detect triggers and perform intelligent text transformations without you ever needing to leave the app you're in.

Key Features:
- AI Capabilities: Query Google Gemini, Cloudflare Workers AI, or any OpenAI-compatible API to fix grammar, translate, adjust tone, or answer questions.
- Global Rewrite: Use custom instructions to transform entire text fields instantly.
- Utility Belt: Offline tools for math calculations, snippets (text expansion), date/time insertion, and password generation.
- Privacy First: Data is sent directly to your chosen API provider. No intermediate servers are used, and text is only processed when you use a trigger.
- Safety: Global undo feature and a history manager to recover original text from the last few minutes.

TypeAssist is open-source and respects your privacy.
